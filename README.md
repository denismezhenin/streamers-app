# streamers-app
Streamer spotlight app where you can like and add streamers.

Client:

- React
- Typescript
- React Router
- React Error Boundary
- React Suspence and Lazy
- Vite
- Mantine

Server:
  - Node.js
  - MongoDB
  - Express
 
Start server:
```
cd server
npm start
```

Start client:

```
cd client
npm run dev
```
